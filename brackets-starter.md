# 補足
## ショートカットキー(カッコ内はMac)

* 行の頭にカーソルを飛ばす

  `Alt + ←` 　(`Cmd + ←`)
* 行の末尾にカーソルを飛ばす

  `Alt + →` 　(`Cmd + →`)
* (行の途中にカーソルがあるとき)次の行にカーソルを飛ばす

  `Ctrl + Enter` 　(`Cmd + Enter`)
* コメントアウト

  `Ctrl + /` 　(`Cmd + /`)

これ以外のショートカットは[こちら](https://qiita.com/assialiholic/items/7683d94108b12f1ca3ad)から
